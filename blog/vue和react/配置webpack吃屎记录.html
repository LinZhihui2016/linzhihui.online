# plugin和loader

这里安装之后，不要急着npm run start，webpack.config.js需要一些修改

html-webpack-plugin 这个插件是用来加载html文件的，默认设置下无法加载到我们index.html的，需要手动配置

module.exports = {
	plugins: [
		//其他plugins
		new HtmlWebpackPlugin({
			template: 'index.html'
		})
	],
}
